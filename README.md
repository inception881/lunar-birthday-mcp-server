### Manual Installation
1. **Clone the Repository**
   ```bash

   ```

2. **Client Configuration**
    ```json
    {
      "mcpServers": {
        "lunar-birthday-server": {
          "command": "path\\to\\python.exe",
          "args": [
            "path\\to\\lunar_birthday_mcp.py"
          ]
        }
      }
    }
    ```
